## 1. What is Sidebar Navigation?

The **Sidebar** in Docusaurus is the **primary navigation system for documentation pages**.
It appears on the **left side** and shows:

- Document hierarchy

- Categories and sub-categories

- Current page context

ğŸ“Œ Sidebars are **docs-specific** (not global like Navbar).

## 2. Where Sidebar is Defined

Sidebar configuration lives in:

```js

sidebars.js

```

Example:

```js

module.exports = {
  tutorialSidebar: [],
};


```

## 3. Types of Sidebar Items

Docusaurus supports **multiple sidebar item types**.

### Doc Item (Single Page)

```js

'intro'

```

Maps to:

```js

docs/intro.md


```

### Category (Group of Docs)

```js

{
  type: 'category',
  label: 'Getting Started',
  items: [
    'intro',
    'installation',
  ],
}

```

ğŸ“Œ Categories can be:

- Collapsible

  label: 'Guides',
  items: [{ type: 'autogenerated', dirName: 'guides' }],

## 4. Full Sidebar Example

```js

module.exports = {
  tutorialSidebar: [
    'intro',
    {
      type: 'category',
      label: 'Fundamentals',
      items: [
        'fundamentals/markdown',
        'fundamentals/mdx',
      ],
    },
    {
      type: 'category',
      label: 'Advanced',
      items: [{ type: 'autogenerated', dirName: 'advanced' }],
    },
  ],
};

```

## 5. Sidebar Ordering Rules

Order is determined by:

1. `position` in front matter

2. Order in `sidebars.js`

3. Folder/file name (if autogenerated)

```md

---
title: Installation
position: 1
---


```

## 6. Hide Sidebar on a Page

```md

---
hide_sidebar: true
---

```

Useful for:

- Landing pages

- Intro pages

## 7. [Navbar](./navbar.md) vs Sidebar

| Feature | Navbar | Sidebar |
|--------|--------|---------|
| Scope | Global | Page-specific |
| Navigation level | Top-level navigation | Section navigation |
| Configuration file | `docusaurus.config.js` | `sidebars.js` |
| Number of items | Few items | Many items |
| Visibility | Visible on all pages | Visible only on docs pages |
| Purpose | Entry points to major sections | Deep navigation within docs |
